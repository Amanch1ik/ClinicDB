# 🚀 WorkFlow KG — Платформа Управления Мобильными Ресурсами

> Маркетплейс краткосрочных задач (гиг-экономика) с гео-контролем, эскроу-платежами и автоматическими выплатами. Ориентирован на рынок Кыргызстана.

---

## 📋 О проекте

**WorkFlow KG** — платформа, где **Заказчики** (бизнесы, ИП, ОсОО) публикуют краткосрочные задания, а **Исполнители** (верифицированные физ. лица) выполняют их с жёстким гео-контролем и мгновенными автоматическими выплатами.

### Ключевые особенности:

- 🔐 **Верификация через MBANK ID** — только подтверждённые исполнители
- 📍 **Гео-фенсинг** — check-in/check-out в радиусе 50м от точки задачи
- 💰 **Escrow-система** — заморозка средств при принятии, автовыплата при завершении
- 🛡️ **Антифрод** — защита от Fake GPS, Root/Jailbreak
- 📊 **Админ-панель** — мониторинг GTV, пользователей и задач

## 🛠 Технологический стек

| Слой              | Технология                     | Назначение                          |
| ----------------- | ------------------------------ | ----------------------------------- |
| **Mobile**        | Flutter (Dart)                 | Единое приложение для iOS и Android |
| **Backend**       | Node.js + Express + TypeScript | REST API                            |
| **ORM**           | Prisma                         | Работа с БД                         |
| **БД (основная)** | PostgreSQL                     | Пользователи, задачи, транзакции    |

| **Авторизация**   | JWT + Refresh Tokens           | Безопасная аутентификация           |
| **Админ-панель**  | React (Vite)                   | Веб-интерфейс администратора        |
| **Дизайн**        | Figma                          | UI/UX макеты                        |

## 🚀 Быстрый старт

### Требования:

- Node.js 18+
- PostgreSQL 15+
- Flutter 3.x
- Dart SDK
- (Опционально) MongoDB 7+

### Backend:

```bash
cd backend
npm install
cp .env.example .env          # Настроить переменные окружения
npx prisma migrate dev        # Создать таблицы БД
npx prisma db seed            # Заполнить тестовыми данными
npm run dev                   # Запуск на http://localhost:3000
```

### Mobile:

```bash
cd mobile
flutter pub get
flutter run                    # Запуск на эмуляторе/устройстве
```

### Admin:

```bash
cd admin
npm install
npm run dev                    # Запуск на http://localhost:5173
```
